Overview

Prayer Time App

This project is a mobile prayer application built with React Native & Expo. It helps users stay connected to their daily prayers by providing timings, reminders, and options for different scholars. The app is designed to be simple, user-friendly, and accessible.

Main Features

Prayer Timings — Displays accurate daily prayer schedules with auto-updates.
Scholar Options — Allows users to choose between different scholars (Hansi & Shafi).
Clean UI — Modern, easy-to-navigate interface optimized for both platforms.
Expo Powered — Cross-platform support for Android and iOS.

This app is ideal for users who want a digital companion for prayer, making it easier to follow and manage their daily routines — whether at home or traveling.
