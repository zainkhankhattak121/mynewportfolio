Data Collection

All hostel data — names, distances from Islamabad, contact numbers, room types, pricing, and amenities — is gathered through a combination of live third-party APIs and automated web scraping. The dataset covers every hostel within a 50 km radius from islamabad, kept current through periodic refreshes.

How It Works

1
Data Collection — APIs and web scraping gather all hostels within 50 km of Islamabad.
2
User Input — User sets university, location, budget, room type, and amenities.
3
Similarity Matching — Collaborative filtering scores each hostel against user preferences and past behaviour.
4
Personalised Results — Top-ranked hostels are returned, unique to each user's profile.
