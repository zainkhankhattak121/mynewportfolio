Overview

LungScan AI — CNN-Based Lung Cancer Detection from CT Scans

LungScan AI analyzes CT scans to detect lung cancer with 96% accuracy, classifying nodules as Malignant or Benign to help radiologists prioritize high-risk cases.

Key Features: CT scan upload, Malignant/Benign classification, nodule localization, confidence scores, clinical recommendations, and performance dashboard.

Built for radiologists, oncologists, and hospitals as an AI-assisted second opinion tool.

How It Works

1
Upload CT Scan — User uploads a lung CT scan image in any supported format
2
Preprocessing — Image is resized, normalised and converted for CNN input layer
3
CNN Inference — Deep convolutional layers extract texture, shape and density features from the nodule
4
Classification — Final dense layer outputs Malignant or Benign with confidence probability
5
Result Display — Annotated scan, classification label, confidence score and clinical recommendation returned
