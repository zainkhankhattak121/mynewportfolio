Overview

Brain Tumor Detection

This project uses machine learning to detect and classify brain tumors from MRI scans. It can identify tumor presence and type with high accuracy and provides explainability via Grad-CAM visualizations. The goal is to make the system accurate and reliable for real-world medical use.

Main Features

→ Tumor Classification — Identifies brain tumor presence and type from MRI scans with 98% accuracy.
→ Grad-CAM Explainability — Visual heatmaps highlight which regions of the MRI drove the model's decision.
→ Advanced CNN Architecture — Custom convolutional neural network trained on balanced MRI datasets.
→ Fast and Efficient — Processes MRI images quickly with an OpenCV preprocessing pipeline.

Useful for medical diagnosis support, radiology assistance, and AI-powered healthcare — helping clinicians detect tumors earlier and more reliably.
